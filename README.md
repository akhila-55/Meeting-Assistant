# 🤖 MeetMate – AI Agent for Smarter Online Meetings

An AI-powered meeting assistant that converts meeting audio into text, summarizes discussions, extracts action items, and automatically creates Google Calendar events using NLP and speech recognition.

---

## 🚀 Features

- 🎙️ Speech-to-Text using Whisper AI
- 📝 Meeting Summarization using BART Transformer
- ✅ Automatic Action Item Extraction
- 📅 Google Calendar Event Creation
- 🧠 NLP-based Meeting Analysis

---

## 🛠️ Tech Stack

- Python
- OpenAI Whisper
- Hugging Face Transformers
- BART (facebook/bart-large-cnn)
- SpaCy
- Google Calendar API
- Google Colab
- PyDub

---

## 📂 Project Workflow

1. Upload meeting audio file
2. Convert MP3 to WAV format
3. Transcribe audio into text
4. Generate meeting summary
5. Extract action items
6. Create Google Calendar event automatically

---

## 📊 Models Used

- Whisper ASR Model
- BART Large CNN
- SpaCy NLP

---

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Install required libraries
3. Upload meeting audio file
4. Run all cells sequentially
5. Authenticate Google Calendar API
6. View generated summary and event
